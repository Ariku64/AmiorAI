# Audit AmiorAI v40.0.6 — exemples de saisie génériques

- Tous les placeholders traduits de l’interface ont été relus.
- Les références personnelles, les jetons identifiables et les prénoms fixes ont été retirés.
- Les scénarios trop spécifiques ont été remplacés par des indications neutres.
- Les versions bureau et mobile ont été harmonisées en FR, EN, ES et DE.
- Les exemples techniques utiles (URL locale, chemins, valeurs numériques) sont conservés.
- Le fichier maître `translations_master.xlsx` a été mis à jour pour éviter toute régression lors d’une régénération des locales.
